# intelli-membrane
Repository for explainable transfer learning in membrane science.

The required conda environment can be installed using the provided env.yml file. The codebase was developed and tested on Linux.

An example prediction script can be run with the 

python model_predict.py

command, which performs rejection prediction for the inputs in example/prediction_input.csv, producing an example/prediction_output.csv file with the predicted rejection values.

The OSN Catalyst is available in the osncat.csv file.
